Simple Nessus

.nessus files simplified

Download as .zip Download as .tar.gz View on GitHub

A Perl script developed to render Tenable Nessus reports more tidy by keeping only the relevant information.

It works with any proper .nessus (v1 and v2) file (tested on Nessus 5.0.0, Web Server Version 3.0.14).

  • Version 0.7
  • Last Modified: 16 April 2012

Getting Started

Simple Nessus is developed on Perl v5.12.3 and requires XML::Simple and Getopt::Long. To install them, open the terminal and run

sudo cpan
install XML::Simple
install Getopt::Long

Finally, to execute it run:

perl simple-nessus.pl {.NESSUS-FILE} {VERSION} [SEVERITY] [OUTPUT]

Options

Choose which file version are you using (mandatory!):

-v1 .nessus v1 file
-v2 .nessus v2 file

Choose which vulnerabilities:

-s L    low, medium, high and critical  (default)
-s M    medium, high and critical
-s H    high and critical

Choose how Simple Nessus returns the information you need:

-o O    STDOUT  (default)
-o T    .txt
-o C    .csv
-o M    .md
